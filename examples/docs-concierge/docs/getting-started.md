# Getting started

Skylark organizes work into workspaces. A workspace holds projects, and each
project holds boards of tasks.

## Creating a workspace

Any signed-in user can create a workspace from the home screen. The creator
becomes the workspace admin automatically. A workspace needs a unique name
within your organization.

## Inviting members

Admins invite members by email from Workspace Settings. Invitees pick a role
at accept time if the invite allows it; otherwise they join as viewers. An
invite expires after 14 days.

## Creating a project

Admins and editors can create projects inside a workspace. Every project
starts with a default board named "Backlog" that you can rename or delete.
